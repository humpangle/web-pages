Cloudflare AWS Load Balancer | Setup Guide

by Nikhath K | May 4, 2022

Cloudflare AWS Load Balancer can be set up with Bobcares by your side. Read to learn more. 

At Bobcares, we offer solutions for every query, big and small, as a part of our AWS Support Services.

Let’s take a look at how our AWS Support Team is ready to help customers with Cloudflare AWS Load Balancer.

How to configure Cloudflare AWS Load Balancer

You’ve come to the right place if you’re seeking a way to take advantage of Cloudflare’s features while also using a load balancer. This guide has been put together by our AWS Support Team to assist you with AWS load balancer Cloudflare.

Part 1:

  1. To begin with, head to the console and create a new load balancer.
  2. Then, choose HTTP, HTTPS as the load balancer type and click Create.
  3. Next, add a name for the load balancer as well as all the listeners we require for our application.
  4. After that, specify the availability zones for the Load Balancer and then click Next: Configure Security Settings.
  5. Next, open a new tab and head to Import Certificate in the console. Here we need to enter the Certificate body, Private Key, and Chain of Certificates. We will get these details in the next step.
    1. Open another tab and log in to the Cloudflare account and choose the domain.
    2. Next, head to Origin Server under SSL/TLS and select Create Certificate.
    3. Then, stick with the default option and click Next.
    4. Now, we will get access to the Origin and Private Key. Copy and paste these details in the Import a new certificate window.
  7. After that, download origin_ca_rsa_root.pem file from this link and open it with a text editor.
  8. Next, copy the contents of the file and paste it into the AWS Certificate Manager window and click Next. Our AWS Support Techs would like to remind you to add a tag name for the certificate as well.
  9. Now head back to the window in Step 4 and reload the available certificates to complete configuring Cloudflare AWS Load Balancer. Here we choose the Cloudflare imported certificate and select Next: Configure Security Groups.
  10. Then it is time to select our default Security Group.
  11. After that, head to Configure Routing section by clicking Next
  12. Here we will create a new target and also add the EC2 instance we to handle other than the new AWS load balancer.
  13. Finally, click Review and Create

Part 2:

After the above process, we have to head to the DNS section of the AWS Load Balancer and copy it. We have to paste this information into the Cloudflare account as a CName record. This completes the process of configuring Cloudflare AWS Load Balancer.

If you run into trouble, drop us a line and our AWS Support Techs will get you back on track.
